# TGP_Cepheids_25
Cepheids Telescope Group Project, 2025/26 academic year
Repository for the group project code.

Intended workflow:
1. Calculate photometric magnitudes
2. Use standard stars to generate airmass calibrations (world coordinate system?)
3. Plot light curve and find period of Cepheid pulsation
4. Use period-luminosity relation to compute absolute magnitude
5. Use distance modulus equation to find distance to Cepheids
